Lab Submissions (Labs 1 - 5)

Lab 1: Modern UI with Compose

Designing the static video layout architecture, component cards, and core player interfaces using declarative Compose blocks.

Lab 2: User Interaction & State

Controlling variable state holders and handling complex user search filtering input attributes reactively.

Reflection: I successfully utilized remember and mutableStateOf to safely manage search text alterations. This completely prevented layout refresh flickering during high-frequency keyboard operations.

Lab 3: Material Design 3

Applying structured Scaffolds, customized deep-blue palettes, dynamic typography weights, and light/dark theme parameters.

Reflection: I configured consistent semantic token palettes matching SDG 4 layouts. This built an eye-friendly viewing architecture that effectively accommodates users during long evening study sessions.

Lab 4: Jetpack Navigation Flow

Setting up explicit multi-screen graph maps and routing data tokens via safe parameters across views.

Reflection: I established a centralized navigation host graph that smoothly handles views transitioning from metadata archives directly down to independent active players while maintaining backstack memory structures.

Lab 5: Offline Room Database

Mapping internal SQLite structures using Data Access Objects to construct persistent reactive repository streams.

Reflection: I successfully deployed a persistent local storage cache using Room. This layer ensures remote students can smoothly browse bookmarked historical video assets even in areas with zero network coverage.
